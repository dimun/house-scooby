# house-scooby
Python based  web scrapper

* Python 3.6.5
* Scrapy

## How to ##
 ```
    python -m virtualenv env
    source env/bin/activate
    pip install -r requirements.txt
    cd scrapper/
    scrapy crawl finca_raiz
 ```
